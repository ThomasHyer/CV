Whats Going on?

D&D is a game where you can be and do pretty whatever you choose.

D&D is a role playing game featuring medevial times. In D&D there is all kinds of races monsters and etc...

In the D&D world you usually create a character using a character sheet, then the DM (Dungeon Master) creates the storyline that you follow although the players can do as they choose, and from time to time ruin the DM's plan so the DM has to make up what to do next.
